# corporate-data-analysis

## 概要（OverView）
架空の企業データをもとに、指標の数値の大きい順に並べ替えた結果と、データの平均値を求めた結果、さらに平均値より数値の低い企業のみ選出する絞り込みの結果を表示するプログラムです。

## 開発プロセス（Development Process）
本プロジェクトでは、設計と実装の役割を明確に分けて開発を行いました。

- システム設計・ロジック構築（自身が担当）：
  「どのような企業データを用意し、どういうステップ（並べ替え ➔ 平均値計算 ➔ 絞り込み）で分析を進めるか」という、システムの骨組みや仕様を徹底的に考案しました。
- コード実装（AIが担当）：
  自身が設計した仕様と要件をAIに指示し、実際のPython（pandas）のコード生成・最適化を任せる形で効率的に開発しました。
  
## 使い方（Usage）
1. Pythonをインストールする
2. ターミナルで以下を実行する
   corporate_fata_analysis1.py
3. 資料となるデータと合わせて該当の項目が表示される

## 学んだこと（what I leaened）
- pandasライブラリの基本的な使い方
- DataFrameによる表形式データの扱い方
- sort_values()による平均値計算
- 条件抽出によるデータ分析
  
## 目的（Perpose）
企業データ分析に必要な絞り出しや並べ替え、平均値の計算方法について理解し、より複雑な分析プログラムの作成に生かすため
